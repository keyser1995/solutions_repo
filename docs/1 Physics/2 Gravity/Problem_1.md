## Note about fractions

Definition of fraction: $\frac{a}{b}$ is a fraction where $a$ is the numerator and $b$ is the denominator.

Example: $\frac{1}{2}$

Example in sympy python:

```python
from sympy import *

a, b = symbols('a b')
a/b
```
